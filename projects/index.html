AI 工具导航站

Astro · TypeScript · Tailwind · MDX · Pagefind

为个人长期维护的 AI 工具索引设计内容模型、分类规则、搜索筛选和详情页结构，让工具导航从链接列表变成有判断力的知识库。

查看项目复盘
  • 问题：普通导航站信息密度高，但缺少真实场景。
  • 方案：每个工具必须包含推荐场景、限制和替代工具。
  • 结果：访问者可以按任务而不是厂商选择工具。

个人知识库自动化

n8n · Notion · Readwise · OpenAI API

把阅读、摘录、AI 总结、标签归档和写作选题连接起来，减少资料散落和重复整理。

查看项目复盘
  • 问题：收藏很多，真正能复用的内容很少。
  • 方案：所有输入都要经过摘要、标签、行动项三个步骤。
  • 结果：博客选题和项目资料可以从同一套内容库里复用。

AI 工作流生成器

React · TypeScript · Local Storage · Prompt 模板

根据用户输入的目标、材料和输出格式，生成可复制的 AI 工作流步骤和 Prompt 草稿。

查看项目复盘

  • 方案：把工作流拆成输入、处理、验证、输出四段。
  • 结果：可把经验转化成可分享的模板。
